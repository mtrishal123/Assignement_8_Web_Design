# CRUD App – REST API Assignment

A simple CRUD web app built using HTML, CSS, and JavaScript with a MockAPI/Beeceptor backend.
The app fetches items from an online API, displays them on the homepage, shows full details on a separate page, and includes a Create Item form that sends POST requests to the API.

Features

Fetch & display list of items from API

View item details (API GET request)

Create new items (API POST request + form validation)

Semantic HTML + simple, usable CSS

All pages work with live API — no static data
